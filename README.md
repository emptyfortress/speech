# Speech-drive

Это прототипы speech-drive. При создании использовались следующие технологии:

- vite + ts
- vue 3
- quasar
- mdi-icon
- custom SvgIcon
- pug, scss

![This is an image](/assets/img/image.png)

### Установка и использование

```
npm install
npm run dev
npm run build
```

### Ветки

Различные варианты будут лежать в соответствующих ветках.
